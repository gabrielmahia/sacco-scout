> **Chagua Sacco** /tʃɑːɡuɑː sɑːkɔ/ — *Kiswahili*: "Choose your SACCO".

## Tools

| Tab | Description |
|-----|-------------|
| 🔍 Find | Filter by county, sector, entry budget |
| ⚖️ Compare | Side-by-side radar for up to 3 SACCOs |
| ❤️ Health | NPFL ratio + Capital Adequacy vs SASRA benchmarks |
| ✅ Eligibility | Match your employer sector to joinable SACCOs |

## Data

**SASRA Annual Supervision Report 2023** (public domain) — 20 seed SACCOs.

SASRA benchmarks: NPFL ≤ 5%, Capital Adequacy ≥ 8%.

Past dividends do not guarantee future returns. Confirm current rates directly with each SACCO.
